?? Desktop Cat Pet

A cute animated desktop pet built with Python and PySide6.

Features
?? Idle, Walk, Run, Jump and Sleep animations
?? Multiple skins (Grey, Orange, Black, White)
?? Motivational speech bubbles
?? Random meow sounds
?? Mail check reminders
?? Saves position between sessions
?? Global shortcut (Ctrl + Alt + C) to show/hide the pet
?? Auto-start support on Windows
Technologies Used
Python
PySide6
PyWin32
Keyboard
Pillow
Installation
pip install -r requirements.txt
python main.py
